# 🟣 Interactive Task Dashboard (Full Stack Project)

## 📌 Project Overview
This is a simple Full Stack Task Dashboard application built as part of the Progree Full Stack Development Internship.

The project allows users to manage their daily tasks using basic CRUD operations (Create, Read, Delete).

---

## ✨ Features
- Add new tasks
- View all tasks
- Delete tasks
- Real-time UI updates
- Frontend and Backend integration using REST API

---

## 🛠 Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Fetch API (async/await)

### Backend
- Node.js
- Express.js
- CORS middleware

## 🚀 How to Run This Project

### 1️⃣ Backend Setup
Open terminal in backend folder and run:

npm install express cors
node server.js

Server will run on:
http://localhost:5000

---

### 2️⃣ Frontend Setup
- Open frontend/index.html in browser
- Start using the Task Dashboard

---

## 🔗 API Endpoints

GET /tasks → Get all tasks  
POST /tasks → Add new task  
DELETE /tasks/:id → Delete task
